# e-plantShopping

e-plantShopping is a React-based web application that allows users to browse plants, view details, and manage a shopping cart.

## Features
- Browse different plant categories
- View plant details
- Add and remove plants from cart
- Calculate total price dynamically
- Responsive user interface

## Technologies Used
- React
- Redux Toolkit
- JavaScript
- HTML
- CSS
- Vite

## How to Run the Project Locally
1. Clone the repository
2. Install dependencies using:
   npm install
3. Run the project using:
   npm run preview
